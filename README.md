## Telegram AI Bot

AI-бот для Telegram на Python.

## Возможности

- ответы на вопросы пользователей;
- работа 24/7 через Railway;
- интеграция с AI API;
- асинхронная архитектура на aiogram.

## Технологии

- Python
- aiogram
- Railway
- GitHub

## Установка

```bash
pip install -r requirements.txt
```

Создайте файл `.env`:

```env
TELEGRAM_BOT_TOKEN=your_token
GROQ_API_KEY=your_key
```

Запуск:

```bash
python bot.py
```
